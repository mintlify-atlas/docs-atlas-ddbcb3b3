## What is marco-code?

marco-code is a minimal implementation of an LLM-powered coding assistant. It demonstrates the core **agent loop pattern** — the iterative process where an AI model:

1. Receives a user prompt
2. Decides which tools to call
3. Executes those tools
4. Feeds results back to the model
5. Repeats until the task is complete

With just three built-in tools (`read`, `write`, `bash`), marco-code can help with a wide range of software engineering tasks.

## Quick Example

```bash
marco-code -p "Create a Python script that prints Hello World"
```

marco-code will:
- Understand your request
- Call the `write` tool to create the file
- Return a confirmation message

## Why marco-code?

Most AI coding assistants are complex systems with thousands of lines of code. marco-code strips away the complexity to reveal the essential pattern:

- **Educational**: See exactly how the agent loop works
- **Minimal**: Under 300 lines of Python code
- **Extensible**: Easy to add custom tools and modify behavior
- **Production-Ready**: Includes proper error handling and timeouts
