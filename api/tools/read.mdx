## Overview

The `read` tool reads and returns the complete contents of a file.

## Function Signature

Defined in `marco_code.py:91-94`:

```python
def read(file_path: str) -> str:
    """Read and return the contents of a file."""
    with open(file_path, "r") as f:
        return f.read()
```

## Parameters

<ParamField path="file_path" type="string" required>
  The path to the file to read. Can be absolute or relative to the working directory.
</ParamField>

## Returns

<ResponseField name="content" type="string">
  The complete contents of the file as a string.
</ResponseField>

## Exceptions

The function may raise standard Python file exceptions:

- **`FileNotFoundError`**: If the specified file does not exist
- **`PermissionError`**: If the process lacks permission to read the file
- **`IsADirectoryError`**: If `file_path` points to a directory instead of a file
- **`UnicodeDecodeError`**: If the file contains invalid UTF-8 characters

These exceptions are caught by the agent loop (`marco_code.py:239-240`) and returned as error messages:

