## Overview

The `main` function implements the core "agent loop" pattern used by AI coding assistants. It repeatedly calls the LLM until the task is complete, executing tool calls along the way.

## Agent Loop Pattern

The agent loop is the core pattern (`marco_code.py:210-216`):

```python
# ==========================================================================
# AGENT LOOP
# This is the core pattern: repeatedly call the LLM until it stops.
# Each iteration, the LLM either:
#   1. Calls one or more tools (we execute them and continue)
#   2. Responds with text and finish_reason="stop" (we exit the loop)
# ==========================================================================
```

## Implementation Details

### Loop Execution

The loop runs for up to `MAX_ITERATIONS` (`marco_code.py:217-252`):

```python
for _iteration in range(MAX_ITERATIONS):
    chat = call_model(messages, TOOL_DEFINITIONS)

    if not chat.choices or len(chat.choices) == 0:
        raise RuntimeError("No choices in response")

    message = chat.choices[0].message
    messages.append(message)

    # If the LLM requested tool calls, execute them
    if message.tool_calls:
        for tool_call in message.tool_calls:
            fn = tool_call.function

            try:
                fn_args = json.loads(fn.arguments)
                print(f"  [tool] {fn.name}({fn_args})", file=sys.stderr)
                result = TOOLS[fn.name](**fn_args)
            except (KeyError, json.JSONDecodeError, TypeError) as e:
                result = f"Error: {type(e).__name__}: {e}"
            except subprocess.TimeoutExpired as e:
                result = f"Error: command timed out after {BASH_TIMEOUT}s"
            except Exception as e:
                result = f"Error executing {fn.name}: {e}"

            messages.append(
                {
                    "role": "tool",
                    "tool_call_id": tool_call.id,
                    "content": result,
                }
            )

    # Exit loop when the LLM is done (no more tool calls)
    if chat.choices[0].finish_reason == "stop":
        break
```

### Error Handling

The loop catches and gracefully handles various exceptions (`marco_code.py:231-240`):

- `KeyError`: Tool not found in `TOOLS` dict
- `json.JSONDecodeError`: Invalid JSON in tool arguments
- `TypeError`: Wrong argument types
- `subprocess.TimeoutExpired`: Bash command timeout
- `Exception`: All other errors

Errors are converted to strings and returned to the LLM as tool results, allowing it to recover or inform the user.

### Maximum Iterations Warning

If the loop exhausts all iterations (`marco_code.py:253-257`):

### Building a Custom Agent Loop

For more control, build your own loop using `call_model`:

```python
import json
from marco_code import call_model, TOOL_DEFINITIONS, TOOLS, _build_system_prompt

def custom_agent(prompt: str, max_iterations: int = 5):
    """Custom agent loop with different configuration."""
    messages = [
        {"role": "system", "content": _build_system_prompt()},
        {"role": "user", "content": prompt},
    ]
    
    for iteration in range(max_iterations):
        print(f"Iteration {iteration + 1}/{max_iterations}")
        
        response = call_model(messages, TOOL_DEFINITIONS)
        message = response.choices[0].message
        messages.append(message)
        
        # Execute tool calls
        if message.tool_calls:
            for tool_call in message.tool_calls:
                fn = tool_call.function
                fn_args = json.loads(fn.arguments)
                
                print(f"Calling: {fn.name}({fn_args})")
                result = TOOLS[fn.name](**fn_args)
                
                messages.append({
                    "role": "tool",
                    "tool_call_id": tool_call.id,
                    "content": result
                })
        
        # Check if done
        if response.choices[0].finish_reason == "stop":
            return message.content
    
    return "Max iterations reached"

# Usage
result = custom_agent("Count lines in *.py files")
print(result)
```

### Intercepting Tool Calls

```python
import json
from marco_code import call_model, TOOL_DEFINITIONS, TOOLS

def agent_with_logging(prompt: str):
    """Agent that logs all tool calls to a file."""
    messages = [
        {"role": "user", "content": prompt}
    ]
    
    tool_log = []
    
    for _ in range(10):
        response = call_model(messages, TOOL_DEFINITIONS)
        message = response.choices[0].message
        messages.append(message)
        
        if message.tool_calls:
            for tool_call in message.tool_calls:
                fn = tool_call.function
                fn_args = json.loads(fn.arguments)
                
                # Log the call
                tool_log.append({
                    "tool": fn.name,
                    "args": fn_args,
                })
                
                # Execute
                result = TOOLS[fn.name](**fn_args)
                
                messages.append({
                    "role": "tool",
                    "tool_call_id": tool_call.id,
                    "content": result
                })
        
        if response.choices[0].finish_reason == "stop":
            break
    
    # Save log
    with open("tool_calls.json", "w") as f:
        json.dump(tool_log, f, indent=2)
    
    return message.content
```

    # LLM's final response
    {
        "role": "assistant",

    }

def _build_system_prompt():
